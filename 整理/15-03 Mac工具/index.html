效率工具

  • iTerm2
  • og-my-zsh
  • Go2Shell: 在当前目录打开iTerm2
  • autojump
  • Alfred

使用工具

  • Homebrew：Mac OS 的包管理工具，可以快速安装各种工具
  • Cakebrew：Homebrew的界面管理工具

  • LaunchRocket: 管理Homebrew安装的服务的软件,启动等。安装方法：

  • libimobiledevice： 提供了很多与iOS交互的工具，例如端口映射查看日志

  • tree: 查看当前目录结构树的命令行工具，brew install tree 安装
  • 010 Editor: 二进制分析工具

其他
