## 使用FFmpeg下载m3u8文件

1. 安装 `homebrew`，已安装的跳过
   ```
   $ ruby -e"$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
   # 安装完成 输入brew检查是否成功
   ```
2. 安装 `ffmpeg`
   ```
   $ brew install ffmpeg
   # 验证是否成功
   $ brew info ffmpeg
   ```
3. 下载m3u8->mp4
   ```
   $ ffmpeg -i 视频地址 [输出的文件名.mp4]
   ```
4. 等待下载完成...
