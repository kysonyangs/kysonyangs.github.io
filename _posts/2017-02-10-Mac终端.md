```
sudo codesign -f -d -s - /Applications/Alfred\ 3.app/Contents/Frameworks/Alfred\ Framework.framework/Versions/A/Alfred\ Framework
```

#### 设置别名
1. 首先，你的别名得不存在，在终端输入你想要设置的别名，提示不能找到命令，则别名不存在，我们可以放心使用。 `alias`可以查看当前已设置的所有的别名
2. 永久化的别名设置
```
vim ~/.bash_profile
// 在最下面添加你想要设置的别名，比如
alias cdg="cd ~/Desktop/GG"
// 然后 wq保存退出，在终端输入
source ~/.bash_profile
// 现在，你可以使用你设置的别名了
```

1. 修改终端的显示成 省略前面的电脑名
